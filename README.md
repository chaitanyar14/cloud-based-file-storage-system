# Cloud-Based File Storage System

## Objective
Build a secure cloud file storage system using AWS S3.

## Services Used
- AWS S3
- IAM
- HTML

## Features
- File upload
- File download
- File versioning
- Public sharing via URL

## Architecture
User → Web Interface → AWS S3 Bucket
